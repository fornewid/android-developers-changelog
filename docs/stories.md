# Developer stories

Developers are finding success with features and best practices for Android, Google Play, and the Jetpack libraries.

App developers are utilizing code samples, guides, and API documentation to[develop for Android](https://developer.android.com/develop).

## Games

Game developers are using tools, downloads, samples, libraries, and best practices to create[games on Android](https://developer.android.com/games).

Developers are utilizing[Google Play Instant](https://developer.android.com/topic/google-play-instant)to enable their users to run an app or game without installing it.
