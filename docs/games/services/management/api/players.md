# Players Stay organized with collections Save and categorize content based on your preferences.




A Player is used to represent a human player in Leaderboards and Achievements.

For a list of [methods](#methods) for this resource, see the end of this page.

## Resource representations

There is no persistent data associated with this resource.

## Methods

[hide](/games/services/management/api/players/hide)
:   Hide the given player's leaderboard scores from the given application. This method is only available to user accounts for your developer console.

[unhide](/games/services/management/api/players/unhide)
:   Unhide the given player's leaderboard scores from the given application. This method is only available to user accounts for your developer console and may take up to a day to take effect.
