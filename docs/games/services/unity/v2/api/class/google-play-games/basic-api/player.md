# GooglePlayGames.BasicApi.Player Class Reference

# GooglePlayGames.BasicApi.Player

Represents a player.

## Summary

A player is different from a participant! The participant is an entity that takes part in a particular match; a[Player](https://developer.android.com/games/services/unity/v2/api/class/google-play-games/basic-api/player#class_google_play_games_1_1_basic_api_1_1_player)is a real-world person (tied to a Games account). The player exists across matches, the Participant only exists in the context of a particular match.
