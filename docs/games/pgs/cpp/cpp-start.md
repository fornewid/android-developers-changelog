# Get started with Play Games Services for C and C++ Stay organized with collections Save and categorize content based on your preferences.




This guide describes how to set up a v2 Native C or C++ game
project to use the Play Games Services v2 Native SDK (beta) and integrate the [Platform authentication](/games/pgs/signin). You must complete this integration before you
can add other Play Games Services [features](/games/pgs/overview#features) to your game or integrate
Play Games Services with your [backend game server](/games/pgs/android/server-access).

## Supported features

The v2 Native SDK (beta) supports the following features:

* [Achievements](/games/pgs/cpp/achievements)
* [Recall](/games/pgs/recall/recall-setup#play-games-services-v2-native-sdk)

## API reference documentation

The header files for the SDK contain reference documentation for the APIs. The
header files are located in the `include` folder in the SDK files, which are
available after you sync your project with SDK repository.

## Requirements

* A game project that uses v2 Native C or C++ as the primary
  programming language.
* A game project and development environment that is set up to use the
  [Gradle](https://docs.gradle.org/current/userguide/userguide.html) build system.
