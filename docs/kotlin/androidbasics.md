# Android Basics with Compose

![learn to build apps in the Android Basics with Compose course](https://developer.android.com/static/images/cluster-illustrations/interop.svg)

This course teaches people with no programming experience how to build simple Android apps with Jetpack Compose. Along the way, you'll learn the fundamentals of programming and the basics of the Kotlin programming language. You'll use Android Studio to build a collection of Android apps that begin to show you what's possible with Android.

[Start the course](https://developer.android.com/courses/android-basics-compose/course)
