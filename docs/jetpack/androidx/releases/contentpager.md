## Declaring dependencies

To add a dependency on ContentPager, you must add the Google Maven repository to your project. Read[Google's Maven repository](https://developer.android.com/studio/build/dependencies#google-maven)for more information.

Add the dependencies for the artifacts you need in the`build.gradle`file for your app or module:  

### Groovy

```groovy
dependencies {
    implementation "androidx.contentpager:contentpager:1.0.0"
}
```

### Kotlin

```kotlin
dependencies {
    implementation("androidx.contentpager:contentpager:1.0.0")
}
```

For more information about dependencies, see[Add build dependencies](https://developer.android.com/studio/build/dependencies).
