## What's in Wear OS 5

power

### Power optimizations

The latest version of Wear OS includes several updates and optimizations to help your app run as smoothly as possible in a more power-efficient way.

mobile\_friendly

### Support for different screen sizes

Wear OS 5 includes features that improve system navigation experiences on devices that contain larger displays. In particular, Wear OS 5 supports a denser app selection screen.

watch

### Updates to Watch Face Format

Bring your creative vision even more to life using an updated version of Watch Face Format. Integrate additional types of complications and data expressions to enrich your watch face experience.
