Before you report a new issue, confirm it isn't already included in either the
library's release notes or the list of reported issues.

To report an issue, click **Create a new issue** (alternatively, you can go to
[Google Issue Tracker](https://issuetracker.google.com/issues?q=status:open+componentid:460472)).

[Create a new issue](https://issuetracker.google.com/issues/new?component=460472&template=1554359)

Be sure to provide all the information requested in the issue template.
To subscribe and vote for issues, click the **Star** icon next to an issue in
the tracker. To learn more, see [Subscribe to an Issue](https://developers.google.com/issue-tracker/guides/subscribe#starring_an_issue).
