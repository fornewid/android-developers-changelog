# Review the testing library Stay organized with collections Save and categorize content based on your preferences.



The Android for Cars [androidx.car.app.testing](/reference/androidx/car/app/testing/package-summary) library, which provides
auxiliary classes that you can use to validate your app's performance in a test
environment. For example, the [`SessionController`](/reference/androidx/car/app/testing/SessionController) lets you simulate a
connection to the host and verify that the correct [`Screen`](/reference/androidx/car/app/Screen) and
[`Template`](/reference/androidx/car/app/model/Template) are created and returned.

To learn more, see [`car-samples`](https://github.com/android/car-samples/tree/main/car_app_library).
