# Now in Android

![](http://developer.android.com/static/images/nia/now_in_android.png)  

### Now in Android App Launched in Play Store

Now in Android is a fully functional Android app built with Kotlin and Jetpack Compose. It models Android design and development best practices and was designed to be a useful reference for developers.  
[Download from Play Store](https://play.google.com/store/apps/details?id=com.google.samples.apps.nowinandroid&pli=1) [View on GitHub](https://github.com/android/nowinandroid)
