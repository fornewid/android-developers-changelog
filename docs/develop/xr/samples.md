### Android XR samples

Explore our collection of sample apps and codelabs for Android XR.[![](http://developer.android.com/static/images/xr/hello-xr-sample.jpg)](https://github.com/android/xr-samples)  

### [Hello Android XR](https://github.com/android/xr-samples)

The Hello Android XR sample app uses the Jetpack XR SDK to provide a straightforward example of the basic functionality afforded to Android apps in Android XR. In the sample you can see an implementation of spatial panels, orbiters, environments, and more.  
[Go to GitHub](https://github.com/android/xr-samples)[![](http://developer.android.com/static/images/xr/prototype-AI-glasses-sample-720.png)](https://github.com/android/ai-samples/tree/prototype-ai-glasses)  

### [Prototype AI Glasses Sample](https://github.com/android/ai-samples/tree/prototype-ai-glasses)

This sample demonstrates how to use the Gemini Live API for real-time, voice-based interactions in a simple to-do app. Users can add, remove, and update tasks by speaking to the app, showcasing a hands-free, conversational user experience powered by the Gemini API.  
[Go to GitHub](https://github.com/android/ai-samples/tree/prototype-ai-glasses)[![](http://developer.android.com/static/images/develop/xr/xr-codelabs.png)](https://github.com/android/xr-codelabs)  

### [Android XR Codelabs](https://github.com/android/xr-codelabs)

Learn how to build immersive apps with Jetpack Compose for XR by following step-by-step instructions.  
[Go to GitHub](https://github.com/android/xr-codelabs)[![](http://developer.android.com/static/images/develop/xr/unity-sample.png)](https://github.com/android/android-xr-unity-package)  

### [Android XR Extensions for Unity Samples](https://github.com/android/android-xr-unity-package)

Learn how to build immersive apps with the Android XR Extensions for Unity. This set of samples contains a standalone Unity project sample for each of the features in this Unity package, including face tracking, HandMesh, object tracking, and more.  
[Go to GitHub](https://github.com/android/android-xr-unity-package)[Go to our Unity quickstart](http://developer.android.com/develop/xr/unity/xr-extensions-quickstart)
