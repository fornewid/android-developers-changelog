### [Build UI with Jetpack Compose Glimmer](http://developer.android.com/develop/xr/jetpack-xr-sdk/jetpack-compose-glimmer)

Jetpack Compose Glimmer is a UI toolkit for building augmented Android XR experiences, optimized for display AI Glasses. Build beautiful, minimal, and comfortable UI for devices that are worn all day.  
[Build UI with Jetpack Compose Glimmer](http://developer.android.com/develop/xr/jetpack-xr-sdk/jetpack-compose-glimmer)[Learn about Jetpack Compose Glimmer components design](http://developer.android.com/design/ui/ai-glasses/guides/components/overview)[![](http://developer.android.com/static/images/develop/xr/jetpack-xr-sdk/conversational.png)](http://developer.android.com/develop/xr/jetpack-xr-sdk/gemini-live)  

### [Build conversational experiences](http://developer.android.com/develop/xr/jetpack-xr-sdk/gemini-live)

Integrate AI and contextual awareness into your app's experiences for AI glasses. Build conversational experiences with the Gemini Live API that enable voice to voice interaction with Gemini models.  
[Integrate with the Gemini Live API](http://developer.android.com/develop/xr/jetpack-xr-sdk/gemini-live)[Learn about designing audio flow \& queues](http://developer.android.com/design/ui/ai-glasses/guides/foundations/get-started#audio-flow-cues)[![](http://developer.android.com/static/images/develop/xr/jetpack-xr-sdk/helpful-information.jpg)](http://developer.android.com/develop/xr/jetpack-xr-sdk/arcore/geospatial)  

### [Deliver helpful information that enhances a user's daily life](http://developer.android.com/develop/xr/jetpack-xr-sdk/arcore/geospatial)

Provide helpful and timely information that keeps the user focused on the world around them while enriching their life. For example, use the Geospatial API together with device pose information through ARCore for Jetpack XR to help a user explore and navigate real-world spaces.  
[Work with geospatial poses](http://developer.android.com/develop/xr/jetpack-xr-sdk/arcore/geospatial)![](http://developer.android.com/static/images/spot-icons/jetpack.svg)  

## Supported tools and technologies

The Jetpack XR SDK includes all the tools you need to build augmented experiences for AI glasses.  

Use Jetpack Compose Glimmer, Android Studio, the emulator, and ARCore for Jetpack XR.  
[Get started](http://developer.android.com/develop/xr/jetpack-xr-sdk/get-started)
