The Jetpack XR SDK consists of the libraries and tools that you need to develop apps for Android XR.

This guide covers the following areas:

- [Installing and configuring Android Studio for XR development](https://developer.android.com/develop/xr/jetpack-xr-sdk/get-studio)
- [Creating an Android XR project](https://developer.android.com/develop/xr/jetpack-xr-sdk/create-project)
- [Setting up the Jetpack XR SDK](https://developer.android.com/develop/xr/jetpack-xr-sdk/set-up-sdk)

This guide assumes you're familiar with the following concepts:

- Downloading and installing applications, such as Android Studio.
- Libraries in[Android Jetpack](https://developer.android.com/jetpack).
