Godot supports Android XR through its OpenXR integration. You can build and
export XR applications for Android using the standard Godot workflow.

## Prerequisites

* **Godot Engine 4.5 or higher**: Verify that you are using a compatible
  version of Godot.
* **Godot OpenXR Vendors Plugin**: This plugin provides necessary extensions
  for Android XR devices.

## Get started

1. **Install the Plugin**: Download and install the [Godot OpenXR Vendors
   Plugin](https://github.com/GodotVR/godot_openxr_vendors) from the Godot Asset Library or GitHub.
2. **Configure Export**: Go to **Project > Export**, add an **Android** preset,
   and verify that **OpenXR** is enabled in the XR Features section.
3. **Learn More**: Follow the official [Godot XR documentation](https://docs.godotengine.org/en/stable/tutorials/xr/index.html)
   for detailed setup instructions.

## Get support

If you encounter issues, see the [support section for Godot](/develop/xr/support#godot) on our support
page for help.
