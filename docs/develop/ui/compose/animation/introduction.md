# Animations in Compose

### [Animations in Compose](http://developer.android.com/develop/ui/compose/animation/quick-guide)

Animations are essential in a modern mobile app in order to realize a smooth and understandable user experience.  
[Quick guide](http://developer.android.com/develop/ui/compose/animation/quick-guide)

## Animation modifiers and composables

[![](http://developer.android.com/static/develop/ui/compose/images/animations/animated_visibility_column.gif)](http://developer.android.com/develop/ui/compose/animation/composables-modifiers#animatedvisibility)  

### [Animate appearance and disappearance](http://developer.android.com/develop/ui/compose/animation/composables-modifiers#animatedvisibility)

The`AnimatedVisibility`composable allows you to hide or show content easily.  
[Learn more](http://developer.android.com/develop/ui/compose/animation/composables-modifiers#animatedvisibility)  
[![](http://developer.android.com/static/develop/ui/compose/images/animations/animated_content_size.gif)](http://developer.android.com/develop/ui/compose/animation/composables-modifiers#animateContentSize)  

### [Animate content size changes](http://developer.android.com/develop/ui/compose/animation/composables-modifiers#animateContentSize)

Use`animateContentSize()`to achieve automatic size change animations.  
[Learn more](http://developer.android.com/develop/ui/compose/animation/composables-modifiers#animateContentSize)  
[![](http://developer.android.com/static/develop/ui/compose/images/animations/animated_content_slower.gif)](http://developer.android.com/develop/ui/compose/animation/composables-modifiers#animatedcontent)  

### [Animate between different composables](http://developer.android.com/develop/ui/compose/animation/composables-modifiers#animatedcontent)

Use`AnimatedContent`to animate between composables that have different content.  
[Learn more](http://developer.android.com/develop/ui/compose/animation/composables-modifiers#animatedcontent)

## Value-based animations

[![](http://developer.android.com/static/develop/ui/compose/images/animations/animated_padding.gif)](http://developer.android.com/develop/ui/compose/animation/value-based#animate-as-state)  

### [Animate a single value](http://developer.android.com/develop/ui/compose/animation/value-based#animate-as-state)

Use`animate*AsState`functions to animate an individual property, such as opacity.  
[Learn more](http://developer.android.com/develop/ui/compose/animation/value-based#animate-as-state)  
[![](http://developer.android.com/static/develop/ui/compose/images/animations/multiple_properties.gif)](http://developer.android.com/develop/ui/compose/animation/quick-guide#concurrent-animations)  

### [Animate multiple values together](http://developer.android.com/develop/ui/compose/animation/quick-guide#concurrent-animations)

Use`Transition`to animate multiple values at once.  
[Learn more](http://developer.android.com/develop/ui/compose/animation/quick-guide#concurrent-animations)  
[![](http://developer.android.com/static/develop/ui/compose/images/animations/animated_forever.gif)](http://developer.android.com/develop/ui/compose/animation/value-based#rememberinfinitetransition)  

### [Animate properties indefinitely](http://developer.android.com/develop/ui/compose/animation/value-based#rememberinfinitetransition)

Use`InfiniteTransition`to animate properties continuously.  
[Learn more](http://developer.android.com/develop/ui/compose/animation/value-based#rememberinfinitetransition)[![](http://developer.android.com/static/images/picto-icons/interoperable.svg)](http://developer.android.com/develop/ui/compose/animation/customize)
