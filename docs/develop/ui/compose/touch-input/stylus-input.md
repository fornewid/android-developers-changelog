# About stylus input Stay organized with collections Save and categorize content based on your preferences.



A stylus enables users to interact with apps comfortably and with pinpoint
accuracy for note‑taking, sketching, working with productivity apps, and
just relaxing and having fun with games and entertainment apps.

## Topics

The stylus input documentation provides guidance on the following topics:

* Handwritten input in text entry fields
* Advanced stylus features such as stylus pressure, orientation, tilt, hover,
  and palm detection
* Stylus support in note-taking apps

## Prerequisites

The stylus input guidance assumes a basic understanding of the following Android
APIs and components:

* Compose and [`WebView`](/reference/kotlin/android/webkit/WebView) text
  fields
* [`EditText`](/reference/kotlin/android/widget/EditText) class
* [`MotionEvent`](/reference/kotlin/android/view/MotionEvent) class
