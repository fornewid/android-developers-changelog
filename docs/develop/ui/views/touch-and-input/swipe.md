Even if your app automatically updates its content on a regular basis, you can also let users
request manual updates. For example, a weather forecasting app can let users refresh the app to get
the latest forecasts on demand. To provide a standard user experience for requesting updates, the
Android platform includes the swipe-to-refresh design pattern, which lets users trigger an update
with a vertical swipe.
