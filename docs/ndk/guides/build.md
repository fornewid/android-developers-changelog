# Build your project Stay organized with collections Save and categorize content based on your preferences.




There are three main ways to build code with the NDK:

* The Make-based [ndk-build](/ndk/guides/ndk-build).
* [CMake](/ndk/guides/cmake).
* [Standalone toolchains](/ndk/guides/standalone_toolchain) for integration
  with other build systems, or use with `configure`-based projects.
