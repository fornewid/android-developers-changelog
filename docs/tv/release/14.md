# Android 14 for TV

This platform update of Android TV OS brings the improvements from Android 13 to compatible consumer devices along with improvements of its own.

Consult the release notes for[Android 13 for TV](https://developer.android.com/about/versions/13/summary)to learn what's new since the previous consumer platform release.

Refer to the[Android 14 documentation](https://developer.android.com/about/versions/14/summary)for a full list of all of the changes included in Android 14. Note that not all those features apply to TV.

Here are some of the highlights of what's new in Android 14 for TV:

- [Picture-in-picture](https://developer.android.com/training/tv/get-started/multitasking)is enabled on select devices that meet our hardware compatibility criteria. To evaluate whether the device supports this feature, query the[PackageManager for the picture-in-picture feature flag](https://developer.android.com/training/tv/get-started/multitasking).

## How to get started

Use the[Android Emulator for TV](https://developer.android.com/training/tv/get-started/create#run-on-a-virtual-device)to try out the new user interface capabilities.
