### Android System Key Verifier

This section provides reference documentation for the
[Key Verifier](https://developer.android.com/privacy-and-security/key-verifier/key-verifier) SDK.

This reference details classes, interfaces, exceptions, and annotations
available to developers working with Key Verifier on Android-powered devices.
