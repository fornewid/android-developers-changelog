### Buttons
Buttons help users initiate actions or flow. Choose from different types of buttons to inform emphasis.](https://developer.android.com/design/ui/tv/guides/components/buttons)[![Cards](https://developer.android.com/static/design/ui/tv/images/landing/cards.svg)
### Cards
Cards contain content and actions about a single subject.](https://developer.android.com/design/ui/tv/guides/components/cards)[![Featured carousel](https://developer.android.com/static/design/ui/tv/images/landing/featured-carousel.svg)
### Featured carousel
Featured carousel highlights a set of media in a full-width container. Guide users towards featured content.](https://developer.android.com/design/ui/tv/guides/components/featured-carousel)[![Immersive list](https://developer.android.com/static/design/ui/tv/images/landing/immersive-list.svg)
### Immersive list
Immersive list highlights a selected item to show immersive content. Use it to emphasis your content.](https://developer.android.com/design/ui/tv/guides/components/immersive-list)[![Lists](https://developer.android.com/static/design/ui/tv/images/landing/lists.svg)
### Lists
Continuous, vertical indexes of text or images. Use lists to help users select from a collection.](https://developer.android.com/design/ui/tv/guides/components/lists)[![Navigation drawer](https://developer.android.com/static/design/ui/tv/images/landing/navigation-drawer.svg)
### Navigation drawer
Navigation drawers provide access to destinations in an app. Drawers are always available at the edge of the screen](https://developer.android.com/design/ui/tv/guides/components/navigation-drawer)[![Tabs](https://developer.android.com/static/design/ui/tv/images/landing/tabs.svg)
### Tabs
Tabs offer a convenient way to switch between primary destinations in an app. Tabs stick to the top of the display.](https://developer.android.com/design/ui/tv/guides/components/tabs)
