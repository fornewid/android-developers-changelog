# Samples | Wear | UI Design

### Wear samples

Explore our collection of design kits and templates for Wear OS.[![](http://developer.android.com/static/images/design/ui/wear/samples/compose-sample.png)](https://github.com/android/wear-os-samples/tree/main/ComposeStarter)  

### [Compose for Wear OS](https://github.com/android/wear-os-samples/tree/main/ComposeStarter)

This sample demonstrates a simple "Hello, World" starter project for using Compose with Wear OS.  
[Go to GitHub](https://github.com/android/wear-os-samples/tree/main/ComposeStarter)[![](http://developer.android.com/static/images/design/ui/wear/samples/golden-tile.png)](https://github.com/android/wear-os-samples/tree/main/WearTilesKotlin)  

### [Golden tile layout](https://github.com/android/wear-os-samples/tree/main/WearTilesKotlin)

This sample showcases the Golden Tile layouts and contains a sample tile built using Tiles Material Components library.  
[Go to GitHub](https://github.com/android/wear-os-samples/tree/main/WearTilesKotlin)[![](http://developer.android.com/static/images/design/ui/wear/samples/horologist.png)](https://github.com/google/horologist)  

### [Horologist library](https://github.com/google/horologist)

A group of libraries that supplement Wear OS with features that are commonly required on watch apps, including the media toolkit and prebuilt composables.  
[Go to GitHub](https://github.com/google/horologist)
