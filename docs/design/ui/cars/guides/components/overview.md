# Components

The following components are used in multiple templates. Here, you'll find guidance on what the component is and how to apply it.

- [Action strip](https://developer.android.com/design/ui/cars/guides/components/action-strip): Put secondary or tertiary actions a tap away.
- [Map action strip](https://developer.android.com/design/ui/cars/guides/components/map-action-strip): Give users access to map interactivity features.
- [Button](https://developer.android.com/design/ui/cars/guides/components/button): Provide access to actions that drivers may need to take.
- [Floating action button (FAB)](https://developer.android.com/design/ui/cars/guides/components/fab): Place the most important actions on the screen a tap away.
- [Header](https://developer.android.com/design/ui/cars/guides/components/header): Help drivers understand which app and what part of the app they're using.
- [Row](https://developer.android.com/design/ui/cars/guides/components/row): Display a small amount of text, plus other options.
