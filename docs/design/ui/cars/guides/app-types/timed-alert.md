# See and respond to a timed alert Stay organized with collections Save and categorize content based on your preferences.



Use a timed alert to let users know how much time they have to respond before a
default action is chosen.

To create a timed alert message, use the Pane view within the
[navigation template](/design/ui/cars/guides/templates/navigation-template), with [timed buttons](/design/ui/cars/guides/components/button) for the default action.

## Sample flow

| User action | Where action is performed | Step count after action |
| --- | --- | --- |
| The user is driving their vehicle. | Navigation template Navigation template during a user's journey | 1 |
| An alert with a timed, default action opens. The user can dismiss the alert or open a map with an alternate route. | Navigation template Navigation template with prompt to resume navigation | 2 |
