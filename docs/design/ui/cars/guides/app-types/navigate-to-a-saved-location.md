# Navigate to a saved location Stay organized with collections Save and categorize content based on your preferences.



You can keep task flows short by presenting location shortcuts in the landing
screen of your navigation apps.

For example, if you include shortcuts such as *Home* and *Work*, users can
start navigation with just one tap.
