# Sectioned Item template Stay organized with collections Save and categorize content based on your preferences.



Make it easy for users to find their favorite media while driving. You can mix
and match lists and grids within the Sectioned Item template to create a
customized browsing structure.

**Note:** With the introduction of the Sectioned Item template, [list](/design/ui/cars/guides/templates/list-template) and
[grid](/design/ui/cars/guides/templates/grid-template) templates are no longer needed. To get the latest updates and
features, move any existing instances of the list or grid templates into the
Sectioned Item template.
