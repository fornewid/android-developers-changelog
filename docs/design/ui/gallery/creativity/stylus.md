## Low latency

Low latency input turns the stylus into a precision tool, providing a natural,
organic writing or drawing experience.

[Learn more](https://developer.android.com/develop/ui/compose/touch-input/stylus-input/advanced-stylus-features#low_latency)
![](https://developer.android.com/static/images/large-screens/gallery/samples/creativity/hover.png)

## Hover

Stylus hover enables contextual information and feedback, such as tooltips, UI
highlighting, and messaging.

[Learn more](https://developer.android.com/develop/ui/compose/touch-input/stylus-input/advanced-stylus-features#hover)

<br />


![](https://developer.android.com/static/images/large-screens/gallery/samples/creativity/palm-rejection.png)

## Palm rejection

Palm detection and rejection eliminate inadvertent screen touches for
comfortable, natural stylus control.

[Learn more](https://developer.android.com/develop/ui/compose/touch-input/stylus-input/advanced-stylus-features#palm_rejection)
![](https://developer.android.com/static/images/large-screens/gallery/samples/creativity/tilt-pressure.png)

## Tilt and pressure

Tilt and pressure support enable advanced input capabilities, such as variation
in stroke width, color saturation, or touch intensity.

[Learn more](https://developer.android.com/develop/ui/compose/touch-input/stylus-input/advanced-stylus-features#pressure)

<br />

## Hover to display

Instantly display media playbacks and file previews with stylus hover.

## Hover to preview

Hover the stylus over the canvas to preview the selected brush size and shape.
