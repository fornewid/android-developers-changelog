Icons must be considered to be glanceable and discernible on glasses displays. Use rounded icon styles with a heavier weight to prevent halation effects.
| **Note:** If you don't need custom iconography, use Material Symbols Rounded.

<br />

![Design elements should be anchored to the bottom of the frame.](https://developer.android.com/static/images/design/ui/glasses/guides/glasses_style_icon_unfilled.png)**Unfilled**Icons should be unfilled to avoid halation, or "light bleed".  
![Design elements should be anchored to the bottom of the frame.](https://developer.android.com/static/images/design/ui/glasses/guides/glasses_style_icon_unfilled.png)**Heavier stroke weight**To ensure readability and harmonize with system typography, system icons should use a heavier weight to avoid thin strokes. Used rounded icon variants.

<br />

| **Note:** Note: For icons, check out Material Symbols. Available on[Google Fonts](https://fonts.google.com/icons).
