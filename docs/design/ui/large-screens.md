## Use proven patterns

Canonical layouts ensure an optimal user experience for all large screen form factors--- tablets, foldables, and ChromeOS devices---as well as supporting all phone sizes. Navigation rail and drawer components provide out-of-the way navigation for UI convenience and control.[![](http://developer.android.com/static/images/design/ui/large-screens/list-detail.svg)](http://developer.android.com/guide/topics/large-screens/large-screen-canonical-layouts#list-detail)  
Canonical

### [List-detail](http://developer.android.com/guide/topics/large-screens/large-screen-canonical-layouts#list-detail)

Enables one-click access to descriptive, explanatory, or other supplementary information for content organized in lists.  
[Learn about list-detail](http://developer.android.com/guide/topics/large-screens/large-screen-canonical-layouts#list-detail)  
[![](http://developer.android.com/static/images/design/ui/large-screens/feed.svg)](http://developer.android.com/guide/topics/large-screens/large-screen-canonical-layouts#feed)  
Canonical

### [Feed](http://developer.android.com/guide/topics/large-screens/large-screen-canonical-layouts#feed)

Arranges content elements in an expansive grid, associating elements by size and placement, drawing attention to elements using proportion and scale.  
[Learn about feed](http://developer.android.com/guide/topics/large-screens/large-screen-canonical-layouts#feed)  
[![](http://developer.android.com/static/images/design/ui/large-screens/supporting-panel.svg)](http://developer.android.com/guide/topics/large-screens/large-screen-canonical-layouts#supporting_pane)  
Canonical

### [Supporting pane](http://developer.android.com/guide/topics/large-screens/large-screen-canonical-layouts#supporting_pane)

Organizes apps into primary and secondary display areas to make tools, options, and settings readily accessible and applicable to the main content.  
[Learn about supporting pane](http://developer.android.com/guide/topics/large-screens/large-screen-canonical-layouts#supporting_pane)  
[![](http://developer.android.com/static/images/design/ui/large-screens/navigation.svg)](http://developer.android.com/guide/topics/large-screens/navigation-for-responsive-uis)  
Canonical

### [Navigation rail and drawer](http://developer.android.com/guide/topics/large-screens/navigation-for-responsive-uis)

Complements the canonical layouts by positioning primary navigation destinations within easy reach while occupying a minimum of screen space.  
[Learn about navigation elements](http://developer.android.com/guide/topics/large-screens/navigation-for-responsive-uis)
