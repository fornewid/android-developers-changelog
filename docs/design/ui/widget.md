# Widgets on Android | UI Design

![](http://developer.android.com/static/images/design/ui/widget/widget-hero.png)  

### Widgets on Android

Add widgets to level up your user experience. Widgets are customizable home screen elements that display a clear and actionable view of an app's content or actions.

## Sizing

[![](http://developer.android.com/static/images/design/ui/widget/sizing-responsive.jpg)](http://developer.android.com/design/ui/mobile/guides/widgets/sizing)  

### [Responsive layouts](http://developer.android.com/design/ui/mobile/guides/widgets/sizing)

Make sure your widget is flexible and consider how your widget could be resized. How might it look in portrait, square, or landscape layout?  
[Learn more](http://developer.android.com/design/ui/mobile/guides/widgets/sizing)[![](http://developer.android.com/static/images/design/ui/widget/sizing-bounds.jpg)](http://developer.android.com/design/ui/mobile/guides/widgets/sizing#fill_the_bounds)  

### [Filling the bounds](http://developer.android.com/design/ui/mobile/guides/widgets/sizing#fill_the_bounds)

Home screen space is precious so it's important to make sure your widget is built to make best use of the space it's been allocated.  
[Learn more](http://developer.android.com/design/ui/mobile/guides/widgets/sizing#fill_the_bounds)
