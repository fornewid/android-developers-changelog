TIER 2 — Adaptive optimized

**Objective:** Make your app [adaptive optimized](/docs/quality-guidelines/adaptive-app-quality/tier-2) by meeting the following keyboard, mouse, and trackpad requirements of the [Adaptive app quality guidelines](/docs/quality-guidelines/adaptive-app-quality):

* [Keyboard\_Navigation](/docs/quality-guidelines/adaptive-app-quality/tier-2#Keyboard_Navigation)
* [Keyboard\_Shortcuts](/docs/quality-guidelines/adaptive-app-quality/tier-2#Keyboard_Shortcuts)
* [Keyboard\_Media\_Playback](/docs/quality-guidelines/adaptive-app-quality/tier-2#Keyboard_Media_Playback)
* [Keyboard\_Send](/docs/quality-guidelines/adaptive-app-quality/tier-2#Keyboard_Send)
* [Context\_Menus](/docs/quality-guidelines/adaptive-app-quality/tier-2#Context_Menus)
* [Content\_Zoom](/docs/quality-guidelines/adaptive-app-quality/tier-2#Content_Zoom)
* [Hover\_States](/docs/quality-guidelines/adaptive-app-quality/tier-2#Hover_States)

Apps optimized for large screen devices provide an enhanced,
accessibility‑compliant user experience by supporting advanced keyboard,
mouse, and trackpad input capabilities, including:

* Keyboard navigation using the `Tab` and arrow keys
* Keyboard shortcuts for actions such as select, cut, copy, paste, undo, and
  redo
* Keyboard control of media playback, such as using the `Spacebar` to
  play and pause media
* Keyboard *send* function in communication apps using the
  `Enter` key
* Context menus accessible by mouse and trackpad right‑click behavior
  (secondary mouse button or secondary tap)
* App content zoom (magnification) using trackpad pinch gestures and the mouse
  scroll wheel with the `Ctrl` or `Control` key
* Hover states for interactive UI elements
