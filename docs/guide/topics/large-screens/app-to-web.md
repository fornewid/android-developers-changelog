# App to web Stay organized with collections Save and categorize content based on your preferences.




![Tier 1 adaptive differentiated icon](/static/images/docs/quality-guidelines/tier-1/tier_1_icon.png)

TIER 1 — Adaptive differentiated

**Objective:** Make your app [adaptive differentiated](/docs/quality-guidelines/adaptive-app-quality/tier-1) by meeting the
[Web\_Transition](/docs/quality-guidelines/adaptive-app-quality/experiences/desktop#Web_Transition) requirement of the [Adaptive app quality guidelines](/docs/quality-guidelines/adaptive-app-quality).

Seamless transitions between app and web content maintain user flow and
application context avoiding fragmented user experiences.
