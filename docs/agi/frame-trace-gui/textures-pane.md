# Textures pane

The**Textures** pane displays all the[texture](https://developer.android.com/agi/frame-trace-gui/texture-pane)resources created up to and including the selected command.

![alt text](https://gapid.dev/images/textures-pane/textures-pane.png "Textures View")

Select a texture resource from the list to see them in[texture view](https://gpuinspector.dev/docs/). Select the**Show deleted textures**checkbox to show textures in the UI even if they have been deleted.
