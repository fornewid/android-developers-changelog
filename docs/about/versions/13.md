## Get started with Android 13

1. **Set up a runtime environment** --- see[Get Android 13](https://developer.android.com/about/versions/13/get)to flash a Google Pixel device or set up an emulator.
2. **Set up Android Studio** --- try the Android 13 SDK and tools. See the[SDK setup](https://developer.android.com/about/versions/13/setup-sdk)page for steps.
3. **Learn about what's new** --- review the[behavior changes for all apps](https://developer.android.com/about/versions/13/behavior-changes-all)and the[behavior changes for apps targeting Android 13](https://developer.android.com/about/versions/13/behavior-changes-13)that might've affected your app.
4. **Test your app** --- run through all flows to look for issues.[Toggle behavior changes](https://developer.android.com/about/versions/13/reference/compat-framework-changes)at runtime to isolate issues.
5. **Update your app**--- target Android 13 if possible, and test with users using beta channels or other groups.
