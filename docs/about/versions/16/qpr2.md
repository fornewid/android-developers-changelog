# Android 16 QPR2

[![](http://developer.android.com/static/about/versions/16/images/android-16-green.svg)](http://developer.android.com/about/versions/16/qpr2/get)  

### [Android 16 QPR2 Beta](http://developer.android.com/about/versions/16/qpr2/get)

The Android 16 QPR2 Beta is now available for development, testing, and feedback! Enroll your device to try it today on Pixel.  
[Get started](http://developer.android.com/about/versions/16/qpr2/get)[Test your app](http://developer.android.com/about/versions/16/qpr2/get#on_emulator)Android 16 QPR2 Beta 3 is now available for you to try with your apps. To get started,[get the beta](https://developer.android.com/about/versions/16/qpr2/get)and[update your tools](https://developer.android.com/about/versions/16/qpr2/setup-sdk). Android 16 Beta is available to users as well as developers, so you should test your app's compatibility and release any needed updates.

## Get started with Android QPR2

1. **Set up a runtime environment** --- see[Get Android 16 QPR2](https://developer.android.com/about/versions/16/qpr2/get)to flash a Google Pixel device or set up an emulator.
2. **Set up Android Studio** --- try the Android 16 SDK and tools. See the[SDK setup](https://developer.android.com/about/versions/16/qpr2/setup-sdk)page for steps.
