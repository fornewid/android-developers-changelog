### Unit 8: Compose with Views

Learn how to use Compose and the older UI toolkit based on Views side-by-side in the same app. In this unit you will learn interoperability APIs and best practices to add a new feature to an existing app in Views, use an existing library that uses Views, or use a UI component that is not yet available in Compose.Pathway 1
