   ## Which line(s) of XML code will produce an error?

   ```kotlin
   1    &ltTextView
   2        android:layout_width="wrap_content"
   3        android:layout_height"wrap_content"
   4        android:padding="8dp"
   5        android:text="@string/title"
   6        android:textSize=18sp /&gt
   ```

   Choose as many answers as you see fit.  
   Line 1 - Missing closing tag \> after`&ltTextView`.  
   Line 3 - Missing = symbol after`android:layout_height`attribute.  
   Line 4 - The`android:padding`attribute does not exist for a`TextView`.  
   Line 5 - You should use`@str/title`to refer to a string resource.  
   Line 6 - Missing quotations around the attribute value`18sp`.
