# TV design review

Optimize a composable for viewing and interaction on a TV screen.

*** ** * ** ***

    This Compose file below defines a custom composable for an Android TV app.
    Optimize it for large screens and remote control interaction.
    Ensure that text is large enough to be read from a distance and that interactive elements have sufficient padding and clear visual feedback on focus.

    $CURRENT_FILE

| To run this prompt in Android Studio, click **Gemini** in the sidebar and paste it in the chat field.   
| To save and retrieve prompts in the Studio IDE, go to **Settings \> Gemini \> Prompt Library**.
