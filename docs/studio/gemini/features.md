Gemini in Android Studio includes features for every step of the development
process. Gemini is a rapidly developing space, so also check Android Studio's
[stable release notes](https://developer.android.com/studio/releases) and
[preview release notes](https://developer.android.com/studio/preview/features) for the latest updates.

| Category | Feature | Description | Docs |
|---|---|---|---|
| Chat | Chat | The original interface for interacting with Gemini. Ask Gemini your Android related questions at any point in your development workflow. | [Chat with Gemini](https://developer.android.com/studio/gemini/chat) |
| Chat | Attach files | Attach files to your query to help Gemini better understand the context for your question and identify what code needs to be updated. | [Attach files](https://developer.android.com/studio/gemini/attach-file) |
| Chat | Attach images | Attach an image to your query to help generate UI, debug UI bugs, document an app's architecture, and more. <br /> In the no-cost tier only. | [Attach images](https://developer.android.com/studio/gemini/attach-image) |
| Chat | Prompt Library (in-IDE) | Save and manage your frequently used prompts and quickly recall them when needed. | [Save and manage prompts with the Prompt Library](https://developer.android.com/studio/gemini/prompt-library) |
| Chat | Rules | Define preferences for Gemini's responses to your queries. Define preferred coding languages, styles, output formats, and more. | [Customize your experience with Rules](https://developer.android.com/studio/gemini/rules) |
| Chat | Prompt Gallery | Browse our collection of prompts to get inspiration on how to incorporate Gemini in to your workflow. | [Prompt gallery](https://developer.android.com/studio/prompt-gallery) |
| Chat | Local third-party models | Choose an LLM locally installed on your computer to power the AI functionality in Android Studio. <br /> In [preview](https://developer.android.com/studio/preview). | [Use a local LLM](https://developer.android.com/studio/gemini/use-a-local-model) |
| Chat | Remote third-party models | Choose an LLM from a remote model provider to power the AI functionality in Android Studio. <br /> In [preview](https://developer.android.com/studio/preview). | [Use a remote LLM](https://developer.android.com/studio/gemini/use-a-remote-model) |
| Agent Mode | Agent Mode | Agent Mode is designed to handle complex, multi-stage development tasks that go beyond what you can experience by just chatting with Gemini. The agent can make changes across multiple files and iteratively fix bugs, accelerating your development process. | [Agent Mode](https://developer.android.com/studio/gemini/agent-mode) |
| Agent Mode | Create a new project | Go from an idea to an app prototype quickly with the help of AI. <br /> In [preview](https://developer.android.com/studio/preview). | [Create a new project with AI](https://developer.android.com/studio/gemini/create-a-new-project-with-ai) |
| Agent Mode | Update dependencies | Update dependencies and iteratively resolve build errors along the way. <br /> In [preview](https://developer.android.com/studio/preview). | [Update dependencies with the Gemini agent](https://developer.android.com/studio/preview/features#update-dependencies) |
| Agent Mode | Add an API key | Add an API key to expand the context window and get even higher quality responses. <br /> Only applicable to users on the default model of the no-cost tier. | [Add your own Gemini API key](https://developer.android.com/studio/gemini/add-api-key) |
| Agent Mode | Add an MCP server | Interact with external tools and extend knowledge and capabilities using the Model Context Protocol (MCP). | [Add an MCP server](https://developer.android.com/studio/gemini/add-mcp-server) |
| Agent Mode | Add a remote MCP server | Connect to an MCP server without having to install and maintain it yourself. <br /> In [preview](https://developer.android.com/studio/preview/features#remote-mcp). | [Add an MCP server](https://developer.android.com/studio/gemini/add-mcp-server) |
| Agent Mode | `AGENTS.md` file support | Define preferences for Gemini's responses to your queries in one or more Markdown files that are part of your codebase. Instructions defined in `AGENTS.md` files are IDE-independent. | [Customize Gemini using `AGENTS.md` files](https://developer.android.com/studio/gemini/agent-files) |
| Agent Mode | Interact with the connected device | The AI agent in Android Studio has access to tools to deploy an app to the connected device, inspect what is currently shown on the screen, take screenshots, check Logcat for errors, and interact with the running app. | [Test and verify changes on a device](https://developer.android.com/studio/gemini/agent-mode#use-cases) |
| Code | Code completion | Gemini offers AI-enabled autocompletion of code in Android Studio that appears as gray italicized text as you type. Code completion saves you time and lets you complete coding projects faster by suggesting full functions. | [Accelerate coding with AI code completion](https://developer.android.com/studio/gemini/code-completion) |
| Code | Code transformation | Prompt Gemini from the code editor for code suggestions to modify, optimize, or add code to your project. | [Transform code](https://developer.android.com/studio/gemini/code-transformation) |
| Code | Rethink variable names | Get suggestions from Gemini for how to rename one or all variables in a file. | [Rethink variable names](https://developer.android.com/studio/gemini/rethink-variable-names) |
| Code | Generate unit test scenarios | Gemini can suggest test scenarios using the context of the code you want to test. When generating unit test scenarios, Gemini includes detailed names and descriptions for your tests, so that you better understand the intention for each suggested test. | [Generate unit test scenarios](https://developer.android.com/studio/gemini/generate-unit-test-scenarios) |
| Code | Journeys with Gemini | Write end-to-end functional tests, called journeys, by describing the steps and assertions using natural language. Gemini converts your steps into actions that Gemini performs on your app. <br /> In [preview](https://developer.android.com/studio/preview/features). | [Journeys for Android Studio](https://developer.android.com/studio/gemini/journeys) |
| Code | Document code | Generate clear and concise documentation for your code with instant drafts that you can refine and perfect. | [Generate documentation for your code](https://developer.android.com/studio/gemini/document-code) |
| Code | Generate commit messages | Gemini can use the context from your code changes and recent past commits to generate a commit message. | [Generate commit messages](https://developer.android.com/studio/gemini/generate-commit-messages) |
| Code | Code customization | Code customization lets you get code suggestions that are based on your organization's private repositories and thus abide by your organization's coding style. Business tier subscribers only. | [Code customization overview](https://developers.google.com/gemini-code-assist/docs/code-customization-overview) |
| Compose | Compose preview generation | Gemini can automatically generate Compose previews, including mock data for preview parameters, for a specific composable or all composables in a file. | [Generate Compose previews](https://developer.android.com/studio/gemini/generate-compose-previews) |
| Compose | Transform UI | Use natural language to update your app UI directly from the Compose preview panel. <br /> Enable through [Studio Labs](https://developer.android.com/studio/gemini/labs). | [Transform UI](https://developer.android.com/studio/gemini/transform-ui) |
| Compose | New UI from a design mock | Generate Compose code directly from a design mock. | [Generate UI with image attachment](https://developer.android.com/studio/gemini/generate-ui-with-images) |
| Compose | Match UI to a target image | Make your UI match a reference design, when you already have an initial UI created. | [Generate UI with image attachment](https://developer.android.com/studio/gemini/generate-ui-with-images) |
| Compose | Fix UI quality issues | Find and fix issues to improve your UI quality and accessibility. | [Find and fix UI quality issues](https://developer.android.com/studio/gemini/generate-ui-with-images#fix-ui-quality) |
| Android-powered integrations | Analyze crashes with App Quality Insights | Use Gemini to analyze your App Quality Insights crash reports, generate insights, provide a crash summary, and (when possible) recommend next steps, including sample code and links to relevant documentation. | [Analyze crashes with App Quality Insights and Gemini](https://developer.android.com/studio/gemini/analyze-crashes-with-aqi) |
| Android-powered integrations | Analyze runtime errors with Logcat | Gemini in Android Studio helps you understand and resolve errors from the Logcat window, streamlining your debugging process. When your app throws an error or exception, click "Ask Gemini" to get immediate explanations and actionable suggestions without leaving the IDE. | [Analyze runtime errors with Logcat and Gemini](https://developer.android.com/studio/gemini/analyze-runtime-errors-with-logcat) |
| Android-powered integrations | Get help with build and sync errors | Gemini understands Gradle build and sync errors. When errors occur, click the "Ask Gemini" link in the build output to help you diagnose and fix the problems. |   |
| Privacy and security | Configure context sharing | Configure which files specifically are shared with Gemini using `.aiexclude` files. | [Configure context sharing](https://developer.android.com/studio/gemini/aiexclude) |
| Privacy and security | Logging | Collect Gemini activity in Cloud Logging, including prompts and reponses and metadata such as lines of code accepted by the user. <br /> Business tier subscribers only. | [Configure Gemini Code Assist Standard and Enterprise logging](https://developers.google.com/gemini-code-assist/docs/configure-logging) |
| Privacy and security | VPC service controls | Establish a secure and controlled environment for coding to protect sensitive data and intellectual property. <br /> Business tier subscribers only. | [Configure VPC Service Controls for Gemini](https://developers.google.com/gemini-code-assist/docs/configure-vpc-service-controls) |
| Privacy and security | Control network access with user domain restrictions | Restrict access to Gemini based on user domains. <br /> Business tier subscribers only. | [Control network access with user domain restrictions](https://developers.google.com/gemini-code-assist/docs/network-access) |
| Other | Productivity metrics | Track your team's usage of Gemini in Android Studio and the impact it's had on your work, including metrics such as the rate of acceptance of code recommendations. <br /> Business tier subscribers only. | [Generate Gemini Code Assist metrics](https://cloud.google.com/gemini/docs/codeassist/generate-metrics) |
