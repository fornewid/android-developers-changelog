# Android Studio for Platform resources

This page contains resources for developers working on Android Studio for Platform.

## Connect with us

To join the Android Studio for Platform developer community, send an email to`asfp-developers+subscribe@android.com`. Once granted access, you can email`asfp-developers@android.com`with questions and feature requests.
