### Android Emulator 36.2.6 Canary

The following updates are included in this release:

- Change the WiFi packets handler to go through netsimd.
- Update libslirp stack
- Support for WiFi Direct (WifiP2p) between multiple AVDs, mDNS (network service discovery), and also hotspot (SoftAp)
