# Fitness guides

![](http://developer.android.com/static/images/health-and-fitness/hero-2.png)  

### Build powerful fitness experiences on Android

Android provides a set of tools and APIs to build innovative fitness apps. From mobile to wearables, you can track user's activity, record data, and provide personalized insights.[![](http://developer.android.com/static/images/picto-icons/wear-os-development.svg)](http://developer.android.com/health-and-fitness/guides/basic-fitness-app/integrate-wear-os)  

### [Extend your app to Wear OS](http://developer.android.com/health-and-fitness/guides/basic-fitness-app/integrate-wear-os)

Enhance your app's health and fitness experience by expanding it onto wearable devices powered by Wear OS. Use Health Services for efficient sensor data access.  
[Learn more](http://developer.android.com/health-and-fitness/guides/basic-fitness-app/integrate-wear-os)[![](http://developer.android.com/static/images/picto-icons/send-data.svg)](http://developer.android.com/health-and-fitness/guides/basic-fitness-app/read-step-count-data)  

### [Measure steps with Sensor Manager](http://developer.android.com/health-and-fitness/guides/basic-fitness-app/read-step-count-data)

Use Sensor Manager to populate steps data in a mobile app. For power-efficiency, consider using the Recording API.  
[Read more](http://developer.android.com/health-and-fitness/guides/basic-fitness-app/read-step-count-data)[![](http://developer.android.com/static/images/picto-icons/health-connect-codelab.svg)](http://developer.android.com/health-and-fitness/guides/recording-api)  

### [Record fitness data on mobile](http://developer.android.com/health-and-fitness/guides/recording-api)

Use the Recording API on mobile to record fitness data from a mobile device in a battery-efficient way. This is a replacement for the deprecated Google Fit Android API.  
[See the guide](http://developer.android.com/health-and-fitness/guides/recording-api)
