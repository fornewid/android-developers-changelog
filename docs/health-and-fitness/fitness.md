### Build powerful fitness experiences on Android

Android provides a set of tools and APIs to build innovative fitness apps. From mobile to wearables, you can track user's activity, record data, and provide personalized insights.

[![](https://developer.android.com/static/images/picto-icons/wear-os-development.svg)](https://developer.android.com/health-and-fitness/guides/basic-fitness-app/integrate-wear-os)

### [Extend your app to Wear OS](https://developer.android.com/health-and-fitness/guides/basic-fitness-app/integrate-wear-os)

Enhance your app's health and fitness experience by expanding it onto wearable devices powered by Wear OS. Use Health Services for efficient sensor data access.

[Learn more](https://developer.android.com/health-and-fitness/guides/basic-fitness-app/integrate-wear-os)

[![](https://developer.android.com/static/images/picto-icons/send-data.svg)](https://developer.android.com/health-and-fitness/guides/basic-fitness-app/read-step-count-data)

### [Measure steps with Sensor Manager](https://developer.android.com/health-and-fitness/guides/basic-fitness-app/read-step-count-data)

Use Sensor Manager to populate steps data in a mobile app. For power-efficiency, consider using the Recording API.

[Read more](https://developer.android.com/health-and-fitness/guides/basic-fitness-app/read-step-count-data)

[![](https://developer.android.com/static/images/picto-icons/health-connect-codelab.svg)](https://developer.android.com/health-and-fitness/guides/recording-api)

### [Record fitness data on mobile](https://developer.android.com/health-and-fitness/guides/recording-api)

Use the Recording API on mobile to record fitness data from a mobile device in a battery-efficient way. This is a replacement for the deprecated Google Fit Android API.

[See the guide](https://developer.android.com/health-and-fitness/guides/recording-api)
