# Support | Health & Fitness Developer Center

### Android Health \& Fitness Support

If you have questions or run into difficulties with Health Connect, use the following resources to find answers and solve problems.

### Give feedback

The Health Connect APIs are evolving and your feedback helps us build a better product. Let us know if you have issues or ideas for improvement.![](http://developer.android.com/static/images/picto-icons/bug.svg)  

### File a bug or feature request

Report issues or send feedback through our public Issue Tracker. Before filing a bug or feature request, check the tracker for similar submissions.
[Go to Issue Tracker](https://issuetracker.google.com/issues/new?component=1676744&template=2072671)  
![](http://developer.android.com/static/images/picto-icons/guidelines.svg)  

### Details to include in bug

When filing a bug, include the following details:

- Steps to reproduce the issue.
- Expected behavior and observed behavior.
- How many times you have reproduced the issue.
- Screenshots or videos, if applicable.

### Looking for more help?

Find highlights, blogs, videos, and other community resources on our Community page.
[Go to Community page](https://developer.android.com/health-and-fitness/community)
