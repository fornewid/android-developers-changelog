## Deepen your understanding

[![](http://developer.android.com/static/images/picto-icons/promotional-2.svg)](http://developer.android.com/guide/topics/sensors/sensors_overview)  

### [Sensor Manager](http://developer.android.com/guide/topics/sensors/sensors_overview)

Use Sensor Manager to create active experiences on mobile. Note: For Wear OS apps, use Health Services instead of Sensor Manager for optimized power efficiency.  
[Learn more](http://developer.android.com/guide/topics/sensors/sensors_overview)  
[![](http://developer.android.com/static/images/spot-icons/permissions.svg)](http://developer.android.com/develop/background-work/services/fgs)  

### [Foreground Services](http://developer.android.com/develop/background-work/services/fgs)

Learn how to use foreground services to keep long running health events active in the background.  
[Learn more](http://developer.android.com/develop/background-work/services/fgs)  
[![](http://developer.android.com/static/images/picto-icons/wearable-arrow.svg)](http://developer.android.com/training/wearables/data/data-layer)  

### [Wearable Data Layer API](http://developer.android.com/training/wearables/data/data-layer)

This API is part of Google Play services, and provides an optional communication channel for apps. Available only for Wear OS watches and paired Android devices.  
[Learn more](http://developer.android.com/training/wearables/data/data-layer)

### [Google Play policy requirements](https://support.google.com/googleplay/android-developer/answer/12991134)

Make sure your app meets Google Play policy requirements related to the collection of health and fitness data.  
[See the requirements](https://support.google.com/googleplay/android-developer/answer/12991134)  
[![](http://developer.android.com/static/images/cluster-illustrations/apis.svg)](http://developer.android.com/guide/health-and-fitness/health-connect-guidelines/migrate/migration-guide)  

### [Google Fit Android API Migration Guide](http://developer.android.com/guide/health-and-fitness/health-connect-guidelines/migrate/migration-guide)

The Google Fit Android API has been marked as deprecated and will be supported until the end of 2024. Migrate to Health Connect or another recommended Android API to preserve your user experience.  
[Read the guide](http://developer.android.com/guide/health-and-fitness/health-connect-guidelines/migrate/migration-guide)
