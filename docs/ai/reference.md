### Google AI Edge SDK Reference

This section provides reference documentation for the[Google AI Edge SDK](https://developer.android.com/ai/gemini-nano).

This reference details classes, interfaces, exceptions, and annotations available to developers working with Gemini Nano on Android-powered devices.
