This section digs into how to prepare your app for multidevice experiences by extending your app to support more devices like tablets, foldables, and Wear OS, voice assistants like[Google Assistant](https://developer.android.com/media/implement/assistant), and[Cast-enabled](https://developers.google.com/cast/docs/overview)devices.

If you're new to media development on Android, consider starting instead with the developer guides for[playback](https://developer.android.com/media/implement/playback-app)and[editing](https://developer.android.com/media/implement/editing-app)for guidance about using the Jetpack Media3 library to implement playback and editing features.
