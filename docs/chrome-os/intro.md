Android developers can distribute apps on Chromebooks through the Google Play
Store. The content in this section shows you how to optimize or build Android
apps for ChromeOS.

If you're interested in web apps on ChromeOS, check out the official ChromeOS
web app [documentation](https://developers.google.com/chromeos/app-development/learn/web).
