 ## Debug tips
 
 Consider the following when debugging:
 
-* If there are double entries in the **Recents** section for a single app,
-  check if you are using the `NEW_TASK` flag inappropriately.
-* If the wrong icon or label appears, verify that each associated non-launcher
-  activity has the same icon, round icon, and label as its parent activity.
-* If the system doesn't launch anything after tapping the entry in the
-  launcher, check Logcat (filtered on `launcher`) for errors, because a
-  trampoline activity can cause this issue.
