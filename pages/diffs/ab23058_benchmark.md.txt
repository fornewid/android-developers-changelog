 ## Version 1.5
 
+### Version 1.5.0-alpha05
+
+March 25, 2026
+
+`androidx.benchmark:benchmark-*:1.5.0-alpha05` is released. Version 1.5.0-alpha05 contains [these commits](https://android.googlesource.com/platform/frameworks/support/+log/1a508f033de883ba2853b9f9ae1853eec7010638..4f1927c2c3b66d0c3a6b9118974d818d2dc5a06a/benchmark).
+
+**Bug Fixes**
+
+- Updated `ArtMetric` to account for top-level packages, especially important when using AGP/R8 9.1+, or `-repackageclasses` global option. Previously, `ArtMetric` would fail to detect these classloads in its `artClassLoad...` measurements. ([If8a35](https://android-review.googlesource.com/#/q/If8a35bdc3e37c707424413d935d1d051dade6eee), [b/487402058](https://issuetracker.google.com/issues/487402058))
+
+**New Features**
+- Added a new instrumentation argument `androidx.benchmark.requireLockedClocks` that allows enabling a check for whether the CPU clocks are locked for rooted devices. ([Id82e8](https://android-review.googlesource.com/#/q/Id82e868b44fb191152427f880e027585210b7472), [b/468039042](https://issuetracker.google.com/issues/468039042))
