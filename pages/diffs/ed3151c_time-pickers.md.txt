 ## Types
 
 There are two types of time picker:
 
-- **Dial**: Lets users set a time by moving a handle around a dial.
-- **Input**: Lets users set a time using their keyboard.
+* **Dial**: Lets users set a time by moving a handle around a dial.
+* **Input**: Lets users set a time using their keyboard.
