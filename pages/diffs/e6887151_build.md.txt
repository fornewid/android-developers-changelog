+    [`minorApiLevel`](https://developer.android.com/reference/tools/gradle-api/current/com/android/build/api/dsl/CompileSdkVersion#minorApiLevel):
+
+    ### Kotlin
+
+    ```kotlin
+        android {

+                }
+            }
+        }

+    ```
+
+    ### Groovy

+                    minorApiLevel = 1
+                }
+            }
+        }
+        
+    ```
