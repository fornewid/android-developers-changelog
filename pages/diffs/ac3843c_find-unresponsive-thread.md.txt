 ### Expensive frame
 
 Rendering too many things in a single frame can cause the main thread to be
 unresponsive for the duration of the frame, such as the following:
 
-* Rendering many unnecessary off-screen items.
-* Using an inefficient algorithm, such as `O(n^2)`, when rendering many UI
-  elements.
+- Rendering many unnecessary off-screen items.
+- Using an inefficient algorithm, such as `O(n^2)`, when rendering many UI elements.
