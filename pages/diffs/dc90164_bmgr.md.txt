+
+`bmgr` is a shell tool you can use to interact with the Backup Manager on Android
+devices version 2.2 (API Level 8) or higher. The tool provides commands to initiate backup and
+restore operations so that you don't need to repeatedly wipe data or take similar intrusive steps
+in order to test your application's backup functionality. The `bmgr` tool supports both
+[Auto Backup](/guide/topics/data/autobackup) and
+[Key/Value Backup](/guide/topics/data/keyvaluebackup).
+
+Note: `bmgr restore` does not work for
+[encrypted backups](/guide/topics/data/autobackup#define-device-conditions).
+
+You run `bmgr` commands on a device via
+[adb shell](/studio/command-line/adb) and then monitor the output of the commands
+with [logcat](/studio/command-line/logcat). For a list and description of
+available commands, run the `bmgr` tool with no arguments. For information on
+triggering backup and restore operations, see
+[Testing Backup and Restore](/guide/topics/data/testingbackup).
+
+For information about adding support for backup in your application, read [Data Backup](/guide/topics/data/backup)
